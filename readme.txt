== Description ==

Standard form calculator converts any number to standard form or scientific notation. It accepts number notation, e-notation, and scientific notation.

== Installation ==

1. Upload the /ci_standard_form_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Standard Form Calculator](https://www.calculator.io/standard-form-calculator/ "Standard Form Calculator Homepage") plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > Standard Form Calculator.
4. And you're good to go!

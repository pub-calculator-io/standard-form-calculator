Standard form calculator converts any number to standard form or scientific notation. It accepts number notation, e-notation, and scientific notation.

[https://www.calculator.io/standard-form-calculator/](https://www.calculator.io/standard-form-calculator/)

== Usage ==

Add the Standard Form Calculator shortcode to your page, post or sidebar:

`[ci_standard_form_calculator]`

Add the following code to your template where you would like the Standard Form Calculator to appear:

`<?php display_ci_standard_form_calculator(); ?>`

== Installation ==

1. Upload the Standard Form Calculator /ci_standard_form_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Standard Form Calculator plugin through the "Plugins" menu in WordPress.
